# Run Worker (Development)
# Dari root project (RECOMMENDED)
go run ./cmd/worker

# Atau dari folder cmd/worker
go run .

# Run Worker (Production dengan PM2)
# 1. Build binary dulu
go build -o sudevwa-worker.exe ./cmd/worker  # Windows
go build -o sudevwa-worker ./cmd/worker      # Ubuntu

# 2. Jalankan dengan PM2
pm2 start cmd/worker/ecosystem.config.js


Hal Penting yang Harus Diperhatikan: Pastikan saat Anda melakukan build, hasilnya ditaruh di folder root agar sesuai dengan isi ecosystem.config.js.

Contoh perintah build-nya:

# Build API
go build -o sudevwa main.go
# Build Worker (Lakukan ini dari root)
go build -o worker cmd/worker/*.go

Setelah di-build, Anda tinggal jalankan:
pm2 start ecosystem.config.js
